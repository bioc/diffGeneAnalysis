biasAdjust(diffGeneAnalysis)R Documentation

baisAdjust performs bias adjustment of normalized data

Description

biasAdjust takes a normalized dataset and applies a multiplicative scalar derived from the data to help account for expression biases. These expression biases can come from many sources including dye bias,hybridization efficacy,changes in personnel,etc.After bias adjustment the data is ready for differential gene analysis.

Usage

biasAdjust(normalized, numSlides)

Arguments

normalized normalized is a matrix of normalized microarray data.The first row consists of headers ans the first column consists of gene names.
numSlides numSlides is the total number of chips of the microarray dataset.

Value

Returns a matrix of bias adjusted normalized data.

Note

biasAdjust takes 2-3 seconds to execute under optimal conditions of size of datasets and speed of the machine..Tested on a Pentium IV 1.6Ghz, 256Mb RAM with 22464 genes and 10 chips it took 12 seconds to execute.
